About

Welcome!

Here you'll be able to find a wide array of bird photographs, always of wild specimens and respecting the natural environment.

This is the first version of the site, and, so far, only the photo gallery is available, but I hope to expand it in the future. To visit the gallery, you can use the menu above or the links in the header and footer.


I have tried to make the site as accessible as possible to users navigating with the keyboard, but screen readers might not work properly yet. In the gallery, when the image carousel is open, navigation using the Tab key is disabled. The arrow keys change the carousel image, the F key opens full screen mode and the Escape key closes the carousel. Also, using the O key opens the carousel on the first photo.


If you have any comments or suggestions, you can contact me via Instagram or e-mail. Likewise, if you find any bugs in the site, I'd be grateful if you could let me know about them. Also, I am stupid and forgot to test on other browsers, so some specific things might not display or work as intended.

Thanks for visiting!
